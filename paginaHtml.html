Tags & dicas HTML 𓂃🖊

HTML é a linguagem usada para criar a estrutura de um site. Com ele, a gente monta tudo: textos, imagens, botões, listas, links… praticamente todo o conteúdo que aparece na tela. As "tags" do HTML indicam o que cada parte da página é — como um título, um parágrafo, uma imagem, etc. Sem o HTML, a página simplesmente não existiria.

<section> 🕮

Descrição: Define uma seção temática dentro do conteúdo, geralmente com título próprio.

Exemplo explicado: Pode ser usada para separar um site em partes como “Sobre nós”, “Serviços” e “Contato”

<header> 🕮

Descrição: Define o cabeçalho de uma página ou seção. Geralmente contém o título e elementos de navegação.

Exemplo: Um topo com logotipo e menu.

<main> 🕮

Descrição: Indica o conteúdo principal único da página.

Exemplo: O corpo central do site, excluindo menus e rodapés.

<article> 🕮

Descrição: Representa um conteúdo independente, como post de blog ou notícia

Exemplo: Postagens individuais em um blog.

<a> 🕮

Descrição: Cria um link clicável para outra página, recurso ou âncora interna.

Exemplo: Texto “Clique aqui” que leva o usuário para outra página quando clicado.

<img> 🕮

Descrição: Insere uma imagem dentro da página.

Exemplo: Exibir uma foto de produto ou logo da empresa.

<ul> 🕮

Descrição: Cria uma lista não ordenada, geralmente com marcadores.

Exemplo: Lista de características de um produto apresentada com pontos.

<ol> 🕮

Descrição: Cria uma lista ordenada, exibindo números ou letras sequenciais.

Exemplo: Passos de um tutorial ou receita em sequência numérica.

<li> 🕮

Descrição: Item individual dentro de listas ordenadas (<ol>) ou não ordenadas (<ul>).

Exemplo: Cada item em uma lista de compras.

<h1 a h6> 🕮

Descrição: Define títulos e subtítulos de diferentes níveis de importância, do mais importante (<h1>) ao menos importante (<h6>).

Exemplo: Usado para organizar hierarquia.

<strong> 🕮

Descrição: Indica que o texto tem forte ênfase, geralmente exibido em negrito.

Exemplo: Usado para destacar palavras importantes em um parágrafo.

<em> 🕮

Descrição: Indica ênfase leve ou entonação diferente no texto, geralmente exibido em itálico.

Exemplo: Usado para destacar uma palavra com significado especial.
