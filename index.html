Glossário Técnico HTML, CSS & JS

Este glossário é um projeto pessoal que criei para me ajudar (e, espero, ajudar você também!) a organizar e consultar os termos mais importantes do HTML e do CSS.

A ideia é ter um lugar simples e direto para tirar as dúvidas. Sabe quando você precisa de uma consulta rápida sobre um código? É pra isso que este glossário existe! Navegue à vontade, confira os exemplos e, quem sabe, pegue uma ou outra dica. A gente aprende muito mais quando compartilha o conhecimento! Espero que este glossário ajude você na sua jornada de aprendizado.
